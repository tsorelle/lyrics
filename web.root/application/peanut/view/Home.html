The purpose of this application is to display lyric sheets in a readable fashion as a reminder prompt for a performer at a song-circle or other event. It is designed to be used on a Kindle Fire but can be viewed on any device.

This diagram explains the buttons on the menu bar of the man page.

Right now, the application only displays my own song collection. If there is interest in a multi-user version I'll support uploading and modifying your own sets and songs. If you'd like this please register on my songs site. Indicate in the "Interests and Instruments" section that you'd like to put your own songs on the "Lyric Sheets" application. I'll let you know when it is released. Register on "Terry's Songs"
